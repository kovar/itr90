How to Use

1. Connect to your gauge

USB (Chrome/Edge only)
Click USB and select your serial port. The ITR 90 streams data continuously at ~50 Hz — no polling needed.
Bridge (any browser)
Start the bridge: uv run bridge.py, then click Bridge. This relays binary data over WebSocket.

2. Toolbar buttons

mbar / Torr / Pa
Switch the display unit on the gauge. The change takes effect immediately.
Degas On / Off
Control the Bayard-Alpert degas function. Only use when pressure is below ~10-5 mbar.
Record / Save
Record begins capturing readings at ~1 Hz. Save stops and downloads a CSV.

3. Chart

Logarithmic Y-axis covers the full range (5×10-10 to 1000 mbar). Window sets visible history. Y Min/Max accepts scientific notation (e.g. 1e-8).

4. Demo mode

Click Demo to simulate a pump-down curve without hardware.

5. Typical workflow

Connect → data streams automatically → click Record to capture → click Save to download CSV.

Running locally

Run uv run serve.py and open http://localhost:8001. Don't open index.html directly.
